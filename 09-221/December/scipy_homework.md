## **Задание 1. Описательная статистика (SciPy + numpy)**

Сгенерируйте выборку из 50 значений, например:

```python
data = np.random.normal(loc=50, scale=10, size=50)
```

### **Задачи:**

1. Используйте `stats.describe()` для получения основных характеристик.
2. Посчитайте отдельно среднее, медиану, минимум и максимум через `scipy.stats`.
3. Найдите 95% доверительный интервал среднего (`stats.t.interval`).

**Подсказка:**
`stats.describe(data)` возвращает число наблюдений, min/max, среднее, дисперсию, эксцесс и асимметрию.
